# salary-calculator

Datum: 2026-07-16

| Thema | Erkenntnis |
|-------|------------|
| Ternary Operator | `bedingung ? wertTrue : wertFalse` ersetzt einfaches if/else in einer Zeile |
| Methoden delegieren | `bonusForProductsSold()` ruft `bonusMultiplier()` auf – Vermeidung von Code-Duplikation |
| Wert kappen (Cap) | `wert >= max ? max : wert` oder `Math.min(wert, max)` begrenzt einen Wert nach oben |
| Ternary vs if | Ternary ist kompakter, aber bei Verschachtelung schwer lesbar – hier klar und lesbar eingesetzt |
